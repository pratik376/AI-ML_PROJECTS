This project is a **Data Analysis Exercise** on Zomato data using **Python**.  
The goal is to explore and analyze Zomato restaurant data to uncover insights about restaurants, cuisines, ratings, and customer preferences.

---

## 📊 Project Overview
- Performed **data cleaning** (handling missing values, duplicates, etc.)  
- Conducted **exploratory data analysis (EDA)** with visualizations  
- Analyzed patterns such as:
  - Most popular cuisines  
  - Distribution of restaurant ratings  
  - Cost vs. rating relationships  
  - City-wise restaurant comparisons  

---

## 🚀 Features
- Insights on **top cuisines** served on Zomato  
- Visualization of **restaurant ratings**  
- Analysis of **cost for two people** across restaurants  
- City-level breakdown of restaurant distribution
