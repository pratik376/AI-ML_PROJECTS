## 📌 Project Overview
This project analyzes Olympic Games data to uncover trends and insights, including medal counts, country performances, and athlete statistics.  
It uses **Python**, **pandas**, **numpy**, and **matplotlib** for data processing and visualization.  

---

## ✨ Features
- Analyze Olympic medal trends by country and year  
- Compare athlete performances  
- Generate visualizations for insights  
- Fully reproducible in Python with a virtual environment  

---

## 🛠 Requirements
- Python 3.x  
- pandas  
- numpy  
- matplotlib  
- scikit-learn (if using ML)  

> **Note:** All dependencies are listed in `requirements.txt`.

## 🚀 How to Run

### Step 1: Clone the Repository
```bash
git clone https://github.com/pratik376/AI-ML_PROJECTS.git
cd AI-ML_PROJECTS/olympic_data_analysis
```

### Step 2: Create Virtual Environment

```bash
Windows: python -m venv .venv
Mac/Linux:python3 -m venv .venv
```

### Step 3: Active virtual Environment

```bash
Windows: .\.venv\Scripts\Activate
Mac/Linux: source .venv/bin/activate
```

### Step 4: Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 5: Run the Jupyter Notebook

```bash
jupyter notebook olympic.ipynb
```

### Step 6: Run the website.py

```bash
python -m streamlit run web.py
```
